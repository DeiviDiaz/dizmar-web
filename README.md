# DIZMAR - Página Web de Catálogo

Sitio web desarrollado para un negocio familiar dedicado a la venta de basecamas, colchones, armarios y muebles para el hogar.

## 🌐 Demo

[Ver sitio web](https://deividiaz.github.io/dizmar-web/)

## 📌 Descripción

Este proyecto consiste en una página web tipo catálogo, diseñada con un enfoque minimalista y elegante, permitiendo a los usuarios visualizar productos y contactarse fácilmente a través de WhatsApp.

## 🎯 Objetivo

* Digitalizar el negocio familiar
* Mostrar productos de forma organizada
* Facilitar el contacto con clientes
* Preparar la base para un futuro e-commerce

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## ⚙️ Funcionalidades

* Navegación con scroll suave
* Catálogo organizado por categorías
* Botones de contacto directo por WhatsApp
* Diseño responsive (adaptable a móviles)

## 🚀 Mejoras futuras

* Implementación de carrito de compras
* Integración de pasarela de pagos
* Panel de administración de productos
* Base de datos
